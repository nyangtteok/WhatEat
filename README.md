# WhatEat - 오늘 뭐 먹지?

먹고 싶은 음식명, 음식 종류, 재료를 입력하면 레시피를 검색하고 랜덤 메뉴를 추천하는 웹 프로젝트입니다.

## 사용 기술
- Frontend: React, Vite, TypeScript, CSS
- Backend: Node.js, Express
- DBMS: SQLite
- Web Storage: localStorage
- Reverse Proxy: Nginx
- Container: Docker, Docker Compose
- CI/CD: GitHub Actions

## 주요 기능
1. 레시피 검색
2. 랜덤 레시피 추천
3. 재료 기반 추천
4. 최근 검색 기록 저장(localStorage)
5. 즐겨찾기 저장(SQLite DBMS)

## 실행 방법
```bash
docker compose up --build
```
브라우저 접속: http://localhost
